الدعم الفني
مركز الدعم والمساعد الذكي
افتح صفحة الدعم لشرح الشحن والسحب أو إرسال مشكلة مع صورة للإدارة.
فتح الدعم
طريقة الشحن
  1. افتح صفحة المحفظة ثم اختر إيداع.
  2. حدد الشبكة المناسبة: TRC20 أو BEP20 أو Polygon.
  3. انسخ عنوان المحفظة وأرسل مبلغ USDT بدقة.
  4. ارفع إثبات الدفع أو اكتب Hash العملية ثم انتظر مراجعة الإدارة.
طريقة السحب
  1. افتح صفحة المحفظة ثم اختر سحب.
  2. اكتب المبلغ وعنوان محفظتك واختر نفس الشبكة.
  3. تأكد من العنوان قبل التأكيد لأن التحويلات لا يمكن عكسها.
  4. طلبات السحب تمر بمراجعة أمنية قبل الإرسال.
كيف يعمل VIP
  1. اشحن رصيدك وانتظر اعتماد الإيداع.
  2. افتح صفحة VIP واختر الخطة المناسبة.
  3. أكد التفعيل وسيتم تسجيل العملية في سجل العمليات.
  4. تابع الرصيد والأرباح من الصفحة الرئيسية والمحفظة.
نظام الإحالة
  1. انسخ رابط الإحالة الخاص بك من صفحة حسابي.
  2. شاركه مع المستخدمين الجدد.
  3. عند إيداع المستخدم المدعو، تظهر عمولة الإحالة في حسابك حسب نظام المنصة.
  4. يمكن متابعة الإحصائيات من قسم إحصائيات الإحالة.
لتسريع حل أي مشكلة
أرسل بريد حسابك، نوع المشكلة، رقم العملية، المبلغ، الشبكة المستخدمة، و Hash التحويل إن وجد. لا ترسل كلمة المرور أو رموز التحقق أو المفاتيح الخاصة لأي شخص.
